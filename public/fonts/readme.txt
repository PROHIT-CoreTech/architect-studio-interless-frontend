Fonts used:
- Cormorant Garamond (Serif): loaded via next/font/google in app/layout.js
- Inter (Sans-Serif): loaded via next/font/google in app/layout.js

If you prefer to load fonts locally in offline mode, download the .woff2 files for Cormorant Garamond and Inter, place them in this folder, and link them using next/font/local.
